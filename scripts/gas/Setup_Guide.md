# Chatwork_Inbound.gs セットアップ手順

> AIEiji秘書 ハイブリッド・ループ Phase 1
> 作成: 2026-04-13 AIEijiSE

---

## 前提条件

- Chatwork APIトークンを取得済みであること（マイチャットに記載あり）
- GoogleドライブにClaudeSync用の00_Inboxフォルダが存在すること

---

## 手順 1: GASプロジェクトを作成

1. ブラウザで https://script.google.com を開く
2. 「新しいプロジェクト」をクリック
3. プロジェクト名を「AIEiji_Chatwork_Secretary」に変更

---

## 手順 2: コードを貼り付け

1. デフォルトの `コード.gs` の中身をすべて削除
2. `Chatwork_Inbound.gs` の内容をすべてコピーして貼り付け
3. Ctrl+S で保存

---

## 手順 3: スクリプトプロパティを設定

1. 左メニューの 歯車アイコン（プロジェクトの設定）をクリック
2. 下部の「スクリプト プロパティ」セクションで「スクリプト プロパティを追加」をクリック
3. 以下の3つを追加:

| プロパティ名 | 値 | 必須 |
|---|---|---|
| `CHATWORK_API_TOKEN` | Chatwork APIトークン（マイチャットに記載の文字列） | 必須 |
| `INBOX_FOLDER_ID` | GoogleドライブのInboxフォルダのID（※下記参照） | 必須 |
| `MY_ACCOUNT_ID` | 自分のChatworkアカウントID（※手順4で取得） | 任意 |

### Googleドライブ フォルダIDの確認方法

1. Googleドライブで、ClaudeSyncの `00_Inbox` フォルダを開く
2. ブラウザのアドレスバーのURLを確認:
   ```
   https://drive.google.com/drive/folders/XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
   ```
3. `folders/` の後の文字列がフォルダID

---

## 手順 4: セットアップ確認（初回のみ）

1. GASエディタに戻る
2. 上部の関数セレクタで `verifySetup` を選択
3. 「実行」ボタンをクリック
4. 初回は「承認が必要です」と表示されるので:
   - 「権限を確認」→ Googleアカウントを選択
   - 「詳細」→「AIEiji_Chatwork_Secretary（安全ではないページ）に移動」
   - 「許可」をクリック
5. 「実行ログ」タブで結果を確認:
   - `認証成功: 三箇栄司 (account_id: XXXXXXX)` と表示されればOK
   - 表示された `account_id` を `MY_ACCOUNT_ID` プロパティに設定

---

## 手順 5: テスト実行

1. 関数セレクタで `testRun` を選択
2. 「実行」をクリック
3. 実行ログで以下を確認:
   - `[INFO] 未読ルーム数: N` → 未読があれば検知成功
   - `[OK] JSON投下完了: chatwork_unread_XXXXXXXX_XXXX.json` → ファイル生成成功
   - `[INFO] 未読なし。処理終了。` → 未読がない場合（正常）
4. Googleドライブの00_Inboxフォルダを開き、JSONファイルが生成されていることを確認
5. ローカルの `C:\ClaudeSync\00_Inbox` にファイルが同期されていることを確認

---

## 手順 6: 定期実行トリガーを設定

1. GASエディタの左メニューで 時計アイコン（トリガー）をクリック
2. 右下の「トリガーを追加」をクリック
3. 以下の設定:

| 項目 | 設定値 |
|---|---|
| 実行する関数 | `checkChatworkUnread` |
| 実行するデプロイ | Head |
| イベントのソース | 時間主導型 |
| 時間ベースのトリガーのタイプ | 分ベースのタイマー |
| 間隔 | 5分おき |

4. 「保存」をクリック

---

## 確認事項

- 5分おきにChatworkの未読をチェックし、未読がある場合のみJSONファイルを生成します
- 同じ時刻（分単位）に2回実行されても重複ファイルは生成されません
- APIのRate Limit（5分あたり300リクエスト）には十分余裕があります
- Chatwork APIトークンはGASのプロパティに保存され、コード内には記載されません

---

## トラブルシューティング

| 症状 | 原因 | 対処 |
|---|---|---|
| `[FATAL] スクリプトプロパティ未設定` | プロパティ未登録 | 手順3をやり直す |
| `[ERROR] API応答 401` | APIトークンが無効 | Chatwork管理画面でトークンを再発行 |
| `[ERROR] DriveApp書き込み失敗` | フォルダIDが不正 | フォルダIDを再確認 |
| JSONが00_Inboxに同期されない | GoogleDrive同期遅延 | 数分待つ / ClaudeSyncの同期状態を確認 |
